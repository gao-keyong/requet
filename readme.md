# 运行方法
1. 将 ReaquestTestSet10 文件夹放入程序目录
2. 运行 preprocess.py
   ```
   python -u "./preprocess.py"
   ```
3. 运行 label.py
   ```
   python -u "./label.py"
   ```
4. 运行 maketraindata.py
   ```
   python -u "./maketraindata.py"
   ```
5. 运行 randomforest_test.py
   ```
   python -u "./randomforest_test.py"
   ```
